# Objectifs
Autoplanning est un outil de planification d'événements (séances, cours) 
basé sur une approche stochastique. Il propose des planifications en tenant compte 
d'un ensemble de contraintes pré-définies.

# Cible client
Il se destine principalement aux
- écoles
- organisateur d'événement
- organisation privée de taille moyenne ou grande
